# odin-rock-paper-scissors

#Rock Paper Scissors

##Project Description
Simple rock,paper,scissors game works with cli in the browser. Game runs for 5 rounds and keeps the track of the score between computer and the player. Also declares the winner.

Computer makes the choices with a Math.random builtin functions. Player chooses every time the browser prompts.

##What I Learned
Worked with functions, conditionals, variables and incrementation. Also the scope for functions.
